# Tech Stack

**Date:** 2026-04-23

## Frontend
- **Language**: TypeScript (`~5.9.3`)
- **Framework**: React (`^19.2.0`), Vite (`^7.2.4`)
- **Styling**: Tailwind CSS (`^3.4.17`), Radix UI, Framer Motion
- **Routing**: `react-router-dom`

## Backend
- **Language**: Python (>=3.10)
- **Framework**: FastAPI (for API), LightRAG (Retrieval-Augmented Generation)
- **Key Libraries**: `nano-vectordb`, `networkx`, `numpy`, `pandas`, `openai`
- **Build System**: `setuptools`

## Configuration
- Environment variables via `.env` files.
- `tsconfig.json` for frontend, `pyproject.toml` for backend.
