> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

## Design System

| Property | Value |
|----------|-------|

| Font | Inter |

---

## Spacing Scale

Declared values (must be multiples of 4 where possible):

| Token | Value | Usage |
|-------|-------|-------|
| 1 | 4px | Inline padding, icon gaps, code block internal padding |
| 2 | 8px | Compact element spacing, markdown paragraph gap |
| 3 | 12px | List item spacing inside markdown |
| 4 | 16px | Chat bubble padding (vertical & horizontal) |
| 6 | 24px | Gap between individual chat messages |
| 10 | 40px | Bottom padding of chat container |

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body (markdown p) | 14px | 400 | 24px |
| Heading 1 (markdown h1) | 24px | 600 | 32px |
| Heading 2 (markdown h2) | 20px | 600 | 28px |
| Code (markdown code) | 13px | 400 | 20px |
| Quote (markdown blockquote) | 14px | 400 | 24px (italicized) |

## Color

| Role | Value | Usage |
|------|-------|-------|
| User Bubble Bg | #262626 | Background for user messages (`semantic/background-elevated`) |
| Assistant Bubble Bg | transparent | Background for assistant messages |
| Assistant Markdown Text | #ffffff | Primary text for assistant responses (`semantic/foreground`) |
| User Text | #ffffff | Primary text for user messages (`semantic/foreground`) |
| Code Block Bg | #000000 | Background for code blocks (`semantic/border-subtle`) |
| Quote Border | #404040 | Left border for blockquotes (`semantic/border`) |
| Typing Indicator | #a1a1a1 | Dots color for loading skeleton (`semantic/foreground-muted`) |
