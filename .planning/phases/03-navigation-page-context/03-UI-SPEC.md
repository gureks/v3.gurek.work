## Color

*These are initial assumptions, to be refined from Figma MCP during implementation.*

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Background, surfaces |
| Secondary (30%) | #F0F0F0 | Cards, sidebar, nav |
| Accent (10%) | #007BFF | Interactive elements, highlights |
| Destructive | #DC3545 | Destructive actions only |
