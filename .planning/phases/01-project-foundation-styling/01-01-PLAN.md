- LAY-02 (UI must be strictly built from scratch based on Figma designs, no Shadcn)

### 4. Setup React Router & Zustand
<read_first>
- frontend/package.json
- frontend/src/main.tsx
- frontend/src/App.tsx
</read_first>
<action>
Run `npm install react-router-dom zustand` in the `frontend` directory.
Create `frontend/src/store/useAppStore.ts` with a Zustand store:
```typescript
import { create } from 'zustand';

interface AppState {
  theme: 'dark' | 'light';
  setTheme: (theme: 'dark' | 'light') => void;
}

export const useAppStore = create<AppState>((set) => ({
  theme: 'dark', // default to dark per design
  setTheme: (theme) => set({ theme }),
}));
```
Update `frontend/src/App.tsx` to include `BrowserRouter`, `Routes`, and a single `Route` for `/` rendering `<div className="bg-background text-foreground min-h-screen">Gurek AI-Twin v3 Foundation</div>`.
Use `useEffect` in `App.tsx` to toggle the `dark` class on `document.documentElement` based on the Zustand `theme` state to fulfill LAY-01.
</action>
<acceptance_criteria>
- `frontend/package.json` dependencies include `zustand` and `react-router-dom`.
- `frontend/src/store/useAppStore.ts` exists and exports `useAppStore`.
- `frontend/src/App.tsx` imports `BrowserRouter` and contains `document.documentElement.classList`.
</acceptance_criteria>
