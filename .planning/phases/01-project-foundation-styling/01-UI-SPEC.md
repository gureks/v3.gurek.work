## Spacing Scale

Declared values (must be multiples of 4 where possible):

| Token | Value | Usage |
|-------|-------|-------|
| 1 | 4px | Inline padding, icon gaps |
| 1-5 | 6px | Tight compact element spacing |
| 2 | 8px | Compact element spacing |
| 2-5 | 10px | Medium-compact element spacing |
| 3 | 12px | List item spacing |
| 4 | 16px | Default element spacing |
| 6 | 24px | Section padding |
| 10 | 40px | Layout gaps |

Exceptions: 6px, 10px values are from the Figma design.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0a0a0a | Background (`semantic/background`) |
| Secondary (30%) | #262626 | Elevated Backgrounds, Sidebar (`semantic/background-elevated`) |
| Accent (10%) | #006eff | Focus states, specific highlights (`semantic/accent`) |
| Foreground | #ffffff | Primary text (`semantic/foreground`) |
| Muted Foreground | #a1a1a1 | Subtitles, placeholders (`semantic/foreground-muted`) |
| Border | #404040 | Standard borders (`semantic/border`) |
| Border Subtle | #000000 | Subtle outlines (`semantic/border-subtle`) |

Accent reserved for: Primary interactive highlights and hover glows.
